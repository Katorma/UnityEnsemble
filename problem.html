Project Summary

Project Sea Phlatt is a top-down, 2D puzzle game that incorporates musical elements into gameplay by utilizing multiple inputs and user interfaces.

Objectives and Tasks

The tasks the team will need to complete to create the game include:

  • Designing an original character
  • Designing levels and puzzles
  • Creating custom pixel art for the character, backgrounds, and objects
  • Design an in-game piano
  • Include sound effects and customizable soundtrack
  • Program the game to be a fully functional RPG

Methods and Approach

The project will be completed using the language C# in conjunction with the software Unity. Unity is a game design platform that features comprehensive organizational tools for assets and code. The team has elected to use Unity due to its wealth of resources and community support.

Most members have not coded in C#, but training in the language and how to use Unity are included in the project scope. There is no anticipated time frame for completion of training, however, project members will be actively learning the features and tools that Unity has to offer.

The team is currently exploring options for soundtrack design, including an open source program called Musescore. Further research and testing is needed before making a final decision.

Currently, the team has chosen a free Pixel Art Editor toolkit for the Unity application in order to create the sprites and visual assets for the game. Other software and editors may be used for refining the assets, but the Editor will be the primary tool used.

Risks and Limitations

  • The team has a limited amount of game development experience
  • Possibility that not all desired features will be able to be included by the deadline
  • Microphone input may be out of scope

Features and Deliverables

  • Multiple interface inputs including accessibility features
  • Evolving soundtrack
  • Dynamic levels
  • Saveable game files

Final Product Goal:

The game will incorporate multiple levels. Each level will reward one with another piece of sheet music. Once a sheet is collected, the player will be able to perform the piece and add it to the already-playing score.
